Win10通过hugo搭建个人博客

Dec 09, 2021
1 minute read

hugo官网下载hugo的windows版本：https://github.com/gohugoio/hugo/releases

将下载的压缩包解压即可使用，解压路径配置到系统的环境变量中即可运行

通过hugo version命令进行版本查看，显示对应版本，说明配置成功

通过hugo new site myblog命令创建文件夹

通过hugo new post/名字.md命令创建文章文件

去到https://themes.gohugo.io/进行hugo主题下载

在themes根目录下执行git clone （url）（url：主题仓库地址）

在myblog根目录下执行hugo server -t （theme） –buildDrafts（theme:下载的主题名字）

启动成功，访问http://localhost:1313/进行本地查看

放到git，新建仓库，仓库名字（owner）.github.io (owner:账户名字)

在myblog根目录下执行hugo –theme=（theme） –baseUrl=“https://（owner）.github.io” –buildDrafts

之后myblog会新增public文件

在public根目录：

  1. git init 初始化
  2. git add . 添加所有文件
  3. git commit -m “第一次提交” 提交文件
  4. git remote add origin （url）（url：仓库地址）
  5. git push -u origin master 提交到远程分支

现在可以查看提交到git上的个人博客了，访问地址https://（owner）.github.io

更新博客操作：

  1. git add -A
  2. git commit -m “更新博客”
  3. git push -f origin master
